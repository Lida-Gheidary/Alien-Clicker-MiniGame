# 👾 Alien Clicker 🧑‍🚀

A fast-paced browser mini-game where you click aliens to score as many points as possible before the timer runs out.

## 🎮 How to Play

1. Enter your name (minimum 3 characters)
2. Click **Start Game**
3. Click the 👾 aliens as fast as you can; each click scores a point
4. You have **60 seconds**. Good luck!
5. When time runs out, your score is submitted and the **Top 10 Scoreboard** appears

## ✨ Features

- 60-second countdown timer
- Live score tracking
- Animated aliens that move around the screen
- Global leaderboard powered by Google Sheets & Zapier
- Top 10 scoreboard displayed as a popup after each game
- Play Again button to instantly restart

## 🛠️ Built With

- HTML, CSS, JavaScript
- [Google Apps Script](https://developers.google.com/apps-script); scoreboard database
- [Zapier](https://zapier.com); score submission webhook
